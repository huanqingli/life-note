## 基本操作的变化
### 声明
新增：let，const  
与 var 对比 let 是块级作用域，且没有变量提升，重复定义会报错。  
与 let 对比 const 声明后无法重新赋值，其声明的对象/数组可以修改。  

支持块级作用域
```js
{
    let food = 'Meow Mix';
};

console.log(food); // Reference Error
//es5要下面这种
(function () {
    var food = 'Meow Mix';
}());

console.log(food); // Reference Error
```
