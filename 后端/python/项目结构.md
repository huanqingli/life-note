## python 项目结构
#### 一个简单的仓库样例
```
README.rst                 
LICENSE                    # 许可证
setup.py                   # 打包和发布管理
requirements.txt           # 开发依赖
sample/__init__.py         # sample 包含核心代码，只有一个文件可以用 sample.py
sample/core.py             
sample/helpers.py          
docs/conf.py               # 参考文档
docs/index.rst             
tests/test_basic.py        # 集合和单元测试
tests/test_advanced.py  
Makefile                   # 常规的管理任务
```
