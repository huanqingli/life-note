#### 三方插件
##### markdown-writer
- `alt-l` : 插入链接
- `alt-i` : 插入图片
- `alt-1` : 插入一级标题，同理2345
- `alt-o` : 有序列表
- `alt-u` : 无需列表
- `alt-t` : 开启任务
- `alt-d` : 完成任务
- `cmd-i` : 斜体
- `cmd-b` : 粗体
- `cmd-h` : 划掉  
- `cmd-'` : 行内代码
- `cmd-;` : 块级代码
- `cmd->` : 引用
- `cmd-j cmd-p` : 跳转到上一个标题
- `cmd-j cmd-n` : 跳转到下一个标题

##### Merge Conflicts
解决 git 合并冲突神器  
`alt-m d` : 检查冲突
